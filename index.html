설정 가이드

3단계로 현대카드 지출을 자동 추적하세요.

1

Google Sheets + Apps Script 설정

1. Google Sheets에서 새 스프레드시트를 만듭니다.
2. 시트 이름을 지출내역으로 변경합니다.
3. 메뉴에서 확장 프로그램 > Apps Script를 클릭합니다.
4. 기본 코드를 모두 지우고, 아래 파일의 내용을 붙여넣습니다. 

📁 apps-script/Code.gs 파일의 전체 내용을 복사하세요

5. 배포 > 새 배포를 클릭합니다.
6. 유형: 웹 앱 선택
7. 실행 사용자: 나
8. 액세스 권한: 모든 사용자
9. 배포 클릭 후 URL을 복사합니다.

2

웹 앱 URL 연결

위에서 복사한 Apps Script 웹 앱 URL을 입력하세요.

3

iPhone 단축어 자동화 설정

iPhone의 단축어 앱에서 자동화를 만들어, 현대카드 알림이 올 때마다 자동으로 데이터를 저장합니다.

1. 단축어 앱 열기
2. 하단 자동화 탭 선택
3. + 버튼 > 개인용 자동화 생성
4. 알림 선택
5. 앱: 지갑 (또는 현대카드 앱) 선택, 포함 내용: 현대카드 입력
6. 즉시 실행 활성화 (확인 없이 자동 실행)
7. 동작 추가에서 아래 단축어를 구성합니다: 

동작 1: URL
  → 여기에 Apps Script URL 입력

동작 2: URL의 콘텐츠 가져오기
  → 방법: POST
  → 본문: JSON
  → 키-값 추가:
    action: addFromNotification
    text: [단축어 입력]

(단축어 입력 = 알림 내용 변수)

참고: iOS 버전에 따라 알림 자동화의 동작이 다를 수 있습니다. 알림 자동화가 제한되는 경우, 수동 입력 버튼을 활용하세요.

*

수동 입력도 가능합니다

자동화가 어려운 경우, 상단의 "+ 수동 입력" 버튼으로 직접 지출을 기록할 수 있습니다. 가맹점 이름을 입력하면 카테고리가 자동 추천됩니다.
